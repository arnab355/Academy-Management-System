# Automated Academy Financial Tracker & Arrears Dashboard

## Key Features Built:
**Dynamic Target Calculation:** Programmed automated cells to dynamically scale monthly revenue targets based on custom individual student dues and real-time new admission counts.
**Row-Insertion Protection:** Structured advanced cell formulas using absolute references (`$`) to ensure total row safety, allowing seamless data entry without breaking core accounting logic.
**Automated Arrears Logic:** Engineered nested conditions to automatically evaluate multi-month payment histories, distinctively distinguishing between old student unpaid tabs, new student exemptions, and temporary student "off" leaves.
